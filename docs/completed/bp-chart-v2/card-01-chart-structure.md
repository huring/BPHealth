# Card 01: Chart Structure

## Objective

Define the data representation for the upgraded blood pressure chart.

## Scope

- Show systolic and diastolic as distinct series
- Use vertical range lines per measurement point
- Optionally include a simple average indicator
- Keep the x-axis time-based
- Work with the current blood pressure data model

## Acceptance Criteria

- The chart structure is clearly defined and implementable
- The design supports the existing reading data without needing new data fields
- Vertical range lines and dual series behavior are specified cleanly

## Status

- Done

## Notes

- Keep the structure simple enough to implement without charting framework complexity
