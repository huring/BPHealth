# Card 03: Visual Style Dark Mode

## Objective

Define a more polished visual style.

## Scope

- Establish a dark-mode-only UI direction
- Define contrast, surfaces, spacing, and component feel
- Keep the UI modern, calm, and readable
- Avoid light colors except where needed for accessibility or data visualization clarity

## Acceptance Criteria

- Dark mode is the default and only intended theme
- Styling direction is clear and consistent
- Readability and contrast remain strong

## Status

- Done

## Notes

- Focus on a coherent visual system rather than decorative complexity
