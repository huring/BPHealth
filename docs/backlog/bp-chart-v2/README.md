# BP Chart V2

## Feature Goal

Create a more premium, mobile-first blood pressure chart that is easier to read at a glance and feels more app-like on dark mode.

## What Is Improved vs the Current Chart

- Adds a richer visual hierarchy with a prominent current value area
- Uses vertical range lines per measurement for a more premium chart feel
- Improves the sense of depth with a darker, more atmospheric background
- Introduces smoother chart interactions and clearer selected-state behavior
- Adds explicit time-range controls above the chart

## Inspiration Elements from the Screenshot

- Large current reading at the top of the chart area
- Dark, glossy background with subtle glow and depth
- Vertical range-style marks for each data point
- Dual-series presentation with clear separation between systolic and diastolic
- Compact time-range chips above the chart
- Minimal clutter and strong hierarchy

## Principles

- Readability first
- Minimal clutter
- Smooth interactions
- Premium but restrained visual treatment
