# Daily Factors

## Feature Goal

Track a few simple daily behaviors alongside blood pressure.

## Why This Matters

These factors help explain patterns around blood pressure without adding much input burden.

## V1 Scope

- Track whether I slept or took a nap that day
- Track whether I drank alcohol that day
- Track how I felt that day:
  - good / productive
  - neutral
  - bad
- Keep input fast and simple

## Out of Scope

- Multi-user support
- Complex mood tracking
- Rich journaling or notes
- Advanced analytics
