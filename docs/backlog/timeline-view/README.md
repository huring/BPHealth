# Timeline View

## Feature Goal

Show blood pressure entries and daily factors together in one chronological view.

## Why This Matters

A combined timeline makes it easier to notice patterns without forcing the user into separate screens.

## V1 Scope

- Combine blood pressure entries
- Combine daily factors
- Show items in chronological order
- Focus on pattern recognition, not complex analysis

## Out of Scope

- Advanced filtering
- Complex grouping controls
- Statistical analysis
- Multi-user support
