# Card 04: Time Range Controls

## Objective

Define the time filtering UX for the chart.

## Scope

- Support 1d, 1w, 1m, 1y, and all time
- Place controls above the chart
- Show a clear active state
- Update the chart smoothly when switching ranges

## Acceptance Criteria

- Switching ranges is fast and intuitive
- There are no confusing intermediate states
- The control row fits naturally into the chart layout

## Notes

- Keep the control set compact and easy to scan on mobile
