# Card 06: Icons, Splash, and Branding

## Objective

Define the assets and presentation needed for an installed-app feel.

## Scope

- App icon requirements
- Splash/loading presentation
- App name and metadata
- Dark-mode-consistent branding

## Acceptance Criteria

- The installed app has the right assets and metadata
- The first impression feels polished

## Notes

- Keep branding simple and consistent with the dark UI direction
- A lightweight branded loading screen is enough for v1
