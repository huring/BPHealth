# Card 01: Combined View

## Objective

Create a single chronological view that combines blood pressure entries and daily factors.

## Scope

- Merge the two data types into one timeline
- Display entries in time order
- Keep the view simple to scan

## Acceptance Criteria

- Blood pressure and daily factor entries appear together
- The view is ordered chronologically
- The presentation stays minimal and readable

## Notes

- Avoid extra navigation or analysis features in this card
