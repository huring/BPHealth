# Blood Pressure Feature

## Feature Goal

Create the first core feature for BPHealth: recording and reviewing blood pressure readings over time.

## Why This Matters

Blood pressure is the primary focus of the app, so this feature is the foundation for the rest of the product experience.

## V1 Scope

- Store blood pressure readings in Supabase
- Let the user add a reading with systolic, diastolic, and measured time
- Show past readings in a simple newest-first list
- Show systolic and diastolic trends in a line chart

## Out of Scope

- Multi-user support
- Authentication work unless required by the Supabase setup
- Editing or deleting entries
- Advanced analytics or derived health insights
- Import/export workflows
