# Card 02: Tag Creation And Reuse

## Objective
Allow creating and reusing tags.

## Scope
- User can create new tags
- App remembers previously used tags
- Tags can be reused easily across measurements

## Acceptance Criteria
- Tag creation is simple
- No duplicate or confusing tag states
- Reuse is fast and intuitive

## Notes
- Prefer lightweight tag discovery over a large management screen.
