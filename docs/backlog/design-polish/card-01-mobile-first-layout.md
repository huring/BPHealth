# Card 01: Mobile First Layout

## Objective

Adapt the app layout for mobile-only use.

## Scope

- Define mobile-first spacing, sizing, navigation, and content hierarchy
- Remove assumptions about desktop layouts
- Prioritize thumb-friendly interaction patterns

## Acceptance Criteria

- Key screens are designed primarily for phone screens
- The layout feels natural on mobile
- No desktop-specific complexity is introduced

## Notes

- Keep this focused on structure and hierarchy, not visual styling details
