# Card 05: Touch Interactions

## Objective

Define better touch behavior across the app.

## Scope

- Tap targets
- Gesture-friendly controls where useful
- Pressed states
- Transitions and feedback
- Keyboard behavior on mobile inputs

## Acceptance Criteria

- Interactions feel responsive and phone-friendly
- Controls are easy to use with fingers
- There are no fiddly or tiny targets

## Notes

- Prefer simple feedback over heavy motion
