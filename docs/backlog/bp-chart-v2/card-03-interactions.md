# Card 03: Interactions

## Objective

Define how the user interacts with the chart.

## Scope

- Tap on data points to reveal values
- Highlight the selected data point or range
- Use smooth state transitions
- Keep interactions lightweight and fast

## Acceptance Criteria

- Interactions feel responsive
- No heavy or slow animations are introduced
- Selected states improve readability instead of adding clutter

## Notes

- Favor simple, mobile-friendly interaction patterns
