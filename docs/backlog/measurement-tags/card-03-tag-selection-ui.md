# Card 03: Tag Selection UI

## Objective
Design a mobile-first UI for selecting and adding tags.

## Scope
- Tags displayed as small chips
- Ability to select existing tags
- Ability to quickly add new tags
- UI should feel modern, fast, and minimal
- Avoid heavy forms

## Acceptance Criteria
- Tag selection is quick and intuitive
- UI works well on mobile
- Chips are clear and readable

## Notes
- This should feel like a lightweight composition step, not a form workflow.
