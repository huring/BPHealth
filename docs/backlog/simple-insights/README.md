# Simple Insights

## Feature Goal

Show very lightweight patterns from the data, without advanced analytics.

## Why This Matters

Simple comparisons can help spot rough relationships without turning the app into a stats tool.

## V1 Scope

- Optional insights only
- Very lightweight pattern summaries
- Example comparisons:
  - average BP on days with alcohol vs without
  - average BP on days slept vs not

## Out of Scope

- Advanced analytics
- Machine learning
- Predictions
- Complex charts or dashboards
